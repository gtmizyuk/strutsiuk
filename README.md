# Йосип Струцюк — сайт письменника

Статичний сайт про волинського письменника Йосипа Струцюка: життєпис,
творчість, аудіозаписи пісень, фотогалерея, відгуки критиків, нагороди.

Це відтворена версія початкового сайту, створеного на платформі Google
Sites учнями та вчителями Рожищенського ліцею №2 для конкурсу
«Літературна візитка краю — 2021» (проєкт посів перше місце серед команд
Волині на обласному етапі). Оригінальний сайт було втрачено без
можливості відновлення; цю версію відтворено на основі архівної копії
головної сторінки та матеріалів, що збереглися в учасників проєкту, зі
збереженням структури первинного сайту.
